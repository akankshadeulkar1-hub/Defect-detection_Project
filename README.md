# 🔍 AI Defect Detection System

An AI-powered web application that detects defects (like cracks, dents, etc.) from images using computer vision.

---

## 🚀 Features

- 📷 Detect defects using **Camera or Image Upload**
- 🧠 AI-based defect classification
- 📊 Confidence score with visual progress bar
- 📂 History sidebar (like ChatGPT)
- 📈 Detailed analysis:
  - Description
  - Severity
  - Recommendation
- 💻 Fully responsive UI

---

### Frontend:

### Backend:
